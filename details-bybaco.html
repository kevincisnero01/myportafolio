SISTEMA WEB - FABRICA BYBACO

Información de Proyecto

  • Categoria: Sistema Web
  • Cliente: Remote - Peru
  • Fecha de Proyecto: 09/19 – 02/20

El objetivo de la aplicación fue crear un software que brindara una vision general de la mercancia textil de medias , asi como gestionar correctamente las etapas de la produccion de las mismas y canalizar pedidos de forma mas eficiente, para asi reducir el margen de error humano y poder dar una respuesta mas precisa a las solicitudes de los clientes.

Para el Back-end se desarrollo en PHP con Laravel, base de datos con PostgretSQL y para el Front-end se desarrollo Bootstrap, CSS y HTML.
