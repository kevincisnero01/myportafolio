API - EMPLEATEYA

Información de Proyecto

  • Categoria: Aplicación Movil
  • Cliente: Remote - Colombia
  • Fecha de Proyecto: 08/19 – 10/19

El objetivo de la aplicación era solucionar la problematica de contratar personal de confianza que realizara algun servicios en nuestro hogar tales como: limpieza, plomeria, electricidad, entre otros, la aplicacion buscaba permitir calificar tanto al personal como a los clientes asi como tambien llevar un registro de pagos.

Para el Back-end se desarrollo una API en PHP con Laravel.
